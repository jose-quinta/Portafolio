# Portafolio - Juan Pérez

Portafolio profesional de **Juan Pérez**, desarrollador Full Stack especializado en .NET, C#, Entity Framework, SQL Server y JavaScript.

## Características

- Diseño responsive (mobile-first)
- SEO optimizado con Open Graph, Twitter Cards y JSON-LD
- Accesibilidad (ARIA, skip link, `:focus-visible`, `prefers-reduced-motion`)
- Imágenes SVG placeholder con degradados
- Menú móvil con animación slide-in y overlay
- Navegación suave con offset por navbar fija
- Active link detection por scroll

## Tecnologías

- HTML5 semántico
- CSS3 (variables CSS, Grid, Flexbox, media queries)
- JavaScript vanilla (ES6+)
